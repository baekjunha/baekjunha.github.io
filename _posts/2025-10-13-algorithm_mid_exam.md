## 1. 분할 정복 알고리즘 (Divide and Conquer)

문제를 하위 단위로 분할하여 각각 해결한 뒤 결과를 결합하는 하향식(Top-down) 전략입니다.

### 1.1 합병 정렬 (Merge Sort)

분할 및 합병 과정을 통해 정렬을 수행합니다.

*   **분할(Divide)**: 배열을 중간 지점을 기준으로 이분할합니다.
*   **정복(Conquer)**: 부분 배열을 재귀적으로 정렬합니다.
*   **결합(Combine)**: 정렬된 배열을 합병하여 최종 결과를 산출합니다.

#### 시간 복잡도 유도
*   **분할 깊이**: 크기 $n$이 1이 될 때까지 이분하므로 깊이는 $\log_2 n$ 입니다.
*   **단계별 연산**: 각 단계의 합병 연산 복잡도는 $O(n)$ 입니다.
*   **최종 복잡도**: $O(n \log n)$ 이며, 데이터 분포와 무관하게 일정합니다.

### 1.2 퀵 정렬 (Quick Sort)

피봇(Pivot)을 기준으로 파티셔닝을 수행합니다.

*   **최선/평균**: 피봇이 배열을 균등하게 분할할 때 $O(n \log n)$ 이 소요됩니다.
*   **최악**: 이미 정렬된 배열에서 편향된 분할이 발생할 경우 $O(n^2)$ 이 소요됩니다.

### 1.3 선택 문제 (Selection Problem)

$k$번째 작은 요소를 찾는 알고리즘입니다. 퀵 정렬의 파티셔닝 원리를 활용하되, 타겟이 포함된 부분만 탐색합니다.

*   **평균 복잡도**: 문제 크기가 기하급수적으로 감소하여 $O(n)$ 에 수렴합니다.
*   **최악 복잡도**: 편향 분할 시 $O(n^2)$ 이 발생합니다.

### 1.4 최근접 점의 쌍 찾기 (Closest Pair)

2차원 평면의 $n$개 점 중 최단 거리의 쌍을 탐색합니다. 분할 정복 적용 시 재귀식 $T(n) = 2T(n/2) + O(n \log n)$ 에 따라 **$O(n \log n)$**의 복잡도를 가집니다.

## 2. 그리디 알고리즘 (Greedy Algorithm)

매 순간 국소적인 최적해를 선택하여 전체 해를 구하는 방식입니다.

### 2.1 최소 신장 트리 (MST)

*   **크러스컬(Kruskal)**: 가중치 순으로 간선을 정렬한 후 사이클을 형성하지 않는 간선을 선택합니다. 복잡도는 **$O(m \log m)$** 입니다.
*   **프림(Prim)**: 특정 정점에서 시작하여 최소 가중치 간선을 선택하며 확장합니다. 우선순위 큐 활용 시 **$O(m \log n)$** 입니다.

### 2.2 최단 경로 (Dijkstra)

특정 정점에서 모든 정점까지의 최단 경로를 산출합니다. 배열 기반 구현 시 **$O(n^2)$** 이 소요됩니다.

### 2.3 부분 배낭 문제 (Fractional Knapsack)

단위 무게당 가치가 높은 순으로 선택합니다. 정렬을 포함하여 **$O(n \log n)$** 이 소요됩니다.

### 2.4 작업 스케줄링 및 허프만 압축

*   **작업 스케줄링**: 시작 시간 기준 정렬 후 할당하며, **$O(n \log n + nm)$** 이 소요됩니다.
*   **허프만 압축**: 빈도수 기준 트리 구성 시 **$O(n \log n)$** 이 소요됩니다.

## 3. 동적 계획법 (Dynamic Programming)

최적 부분 구조를 가진 문제를 하위 문제의 해를 저장(Memoization)하여 재사용함으로써 해결합니다.

### 3.1 모든 쌍 최단 경로 (Floyd-Warshall)

모든 정점 쌍 사이의 최단 거리를 산출합니다. 3중 반복문 구조에 따라 **$O(n^3)$** 이 소요됩니다.

### 3.2 연속 행렬 곱셈 (Chained Matrix Multiplication)

최소 연산 횟수를 보장하는 행렬 곱셈 순서를 탐색합니다. **$O(n^3)$** 의 복잡도를 가집니다.

### 3.3 편집 거리 (Edit Distance)

문자열 변환을 위한 최소 연산 횟수를 산출합니다. $m \times n$ 테이블 구성으로 **$O(mn)$** 이 소요됩니다.

### 3.4 0-1 배낭 문제 (Knapsack Problem)

분할 불가능한 경우의 배낭 문제로, 물건 개수 $n$과 용량 $C$에 대해 **$O(nC)$** 의 의사 다항 시간 복잡도를 가집니다.
