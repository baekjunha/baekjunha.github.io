## 2. 탐색 패턴 분석

### 2.1. 클래스 기반 전체 추출

특정 클래스를 지정하여 해당 속성을 가진 모든 요소를 수집합니다.

```python
import requests
from bs4 import BeautifulSoup

res = requests.get('https://davelee-fun.github.io/blog/crawl_test')
soup = BeautifulSoup(res.content, 'html.parser')

# 'course' 클래스를 포함한 모든 <li> 태그 추출
titles = soup.find_all('li', 'course') 
for title in titles:
    print(title.get_text())
```

### 2.2. 컨테이너 기반 필터링

특정 부모 요소를 식별한 후, 해당 범위 내에서 하위 요소를 탐색하는 이중 필터링을 적용하여 정확도를 높입니다.

```python
# 1. 특정 <ul> 컨테이너 식별
section = soup.find('ul', id='hobby_course_list')

# 2. 컨테이너 내부의 <li class="course"> 탐색
titles = section.find_all('li', 'course')
for title in titles:
    print(title.get_text())
```

### 2.3. 데이터 정제 및 가공

추출된 데이터의 텍스트를 처리하여 필요한 정보만 필터링합니다.

```python
section = soup.find('ul', id='dev_course_list')
titles = section.find_all('li', 'course')

for index, title in enumerate(titles):
    raw_text = title.get_text()
    # 문자열 분리 및 공백 제거를 통한 데이터 정제
    refined_text = raw_text.split('-')[-1].split('[')[0].strip()
    
    print(f"{index + 1}. {refined_text}")
```

## 3. BeautifulSoup 주요 기능 요약

*   **`find()`**: 조건에 부합하는 첫 번째 요소를 반환합니다.
*   **`find_all()`**: 조건에 부합하는 모든 요소를 리스트 형태로 반환합니다.
*   **속성 지정**: `class_` 매개변수를 사용하여 클래스 속성을 명시적으로 타겟팅할 수 있습니다.

## 4. CSS Selector 활용

`select()` 메서드는 CSS 선택자 문법을 사용하여 요소를 정밀하게 추출할 수 있는 기능을 제공합니다.

### 4.1. 주요 메서드
*   **`select()`**: 선택자에 매칭되는 모든 요소를 리스트로 반환합니다.
*   **`select_one()`**: 선택자에 매칭되는 첫 번째 요소만 반환합니다.

### 4.2. 선택자 문법 규칙

*   **태그 선택**: `soup.select('li')`
*   **자손 선택 (Descendant)**: `soup.select('body h1')` (공백 사용)
*   **자식 선택 (Child)**: `soup.select('ul > li')` (`>` 사용)
*   **클래스 및 ID 선택**: `.class`, `#id`
*   **속성 조합**: `li.course.paid` (AND 조건)
*   **복합 계층 구조**: `ul#hobby_course_list li.course`

---

## 5. CSS Selector의 이점

`select()` 메서드는 `find_all()`에 비해 문법이 간결하며, 브라우저 개발자 도구의 'Copy Selector' 기능을 활용할 수 있어 복잡한 DOM 구조를 처리할 때 효율적입니다.

---

## 6. CSS Selector 요약 가이드

| 문법 | 의미 | 예시 |
| :--- | :--- | :--- |
| `'tag'` | 특정 태그명 전체 | `'li'` |
| `'.class'` | 특정 클래스명 | `'.course'` |
| `'#id'` | 특정 고유 ID | `'#start'` |
| `'A > B'` | A의 직계 자식 B | `'ul > li'` |
| `'A.class'`| 태그와 클래스 조합 | `'li.course.paid'` |
| `'A#id B.c'`| ID 기반 자손 탐색 | `'ul#hobby_course_list li.course'` |
