# 카메라의 정의와 구성  

**자료 출처**: [LearnOpenGL](https://learnopengl.com/)

## 1. 카메라 좌표계 정의

카메라(View Space)를 정의하기 위해서는 `위치(Position)`, `시선 방향(Direction)`, `우측 벡터(Right Vector)`, `상단 벡터(Up Vector)`가 필요하다.

<img src="https://learnopengl.com/img/getting-started/camera_axes.png" style="background-color:white;" alt="Camera coordinate axes">  

### 위치 (Camera Position)
월드 공간 내에서 카메라의 좌표를 설정한다. 카메라가 원점으로부터 $+Z$축 방향(사용자 방향)으로 배치되도록 설정하는 예시는 다음과 같다.

```cpp
glm::vec3 cameraPos = glm::vec3(0.0f, 0.0f, 3.0f);
```

### 시선 방향 (Camera Direction)
카메라가 바라보는 방향 벡터를 산출한다. 카메라가 원점을 응시한다고 가정할 때, 원점 벡터에서 카메라 위치 벡터를 뺀 결과값을 정규화하여 방향 벡터를 구한다. OpenGL은 오른손 좌표계를 사용하므로, 카메라의 정면은 $-Z$ 방향을 향하게 된다.

```cpp
glm::vec3 cameraTarget = glm::vec3(0.0f, 0.0f, 0.0f);
glm::vec3 cameraDirection = glm::normalize(cameraPos - cameraTarget);
```

### 우측 및 상단 벡터 (Right & Up Vector)
카메라 좌표계의 $X, Y$축을 정의한다. 세계의 상단 벡터($(0, 1, 0)$)와 카메라 방향 벡터의 외적을 통해 우측 벡터를 구하고, 방향 벡터와 우측 벡터의 외적을 통해 최종 상단 벡터를 산출한다.

```cpp
glm::vec3 up = glm::vec3(0.0f, 1.0f, 0.0f); 
glm::vec3 cameraRight = glm::normalize(glm::cross(up, cameraDirection));
glm::vec3 cameraUp = glm::cross(cameraDirection, cameraRight);
```

## 2. LookAt 행렬

산출된 위치, 방향, 우측, 상단 벡터를 조합하여 **LookAt 행렬**을 생성할 수 있다. 이 행렬은 월드 좌표계의 데이터를 카메라 좌표계(View Space)로 변환하는 뷰 행렬(View Matrix)의 역할을 수행한다.

![alt text](/images/LookAtMat.png)  
_(R: Right, U: Up, D: Direction, P: Position)_

GLM 라이브러리는 이 연산을 자동화하는 `glm::lookAt` 함수를 제공한다.

```cpp
glm::mat4 view;
view = glm::lookAt(
    glm::vec3(0.0f, 0.0f, 3.0f), // 위치
    glm::vec3(0.0f, 0.0f, 0.0f), // 타겟
    glm::vec3(0.0f, 1.0f, 0.0f)  // 상단 벡터
);
```

## 3. 카메라 애니메이션 및 입력 처리

### 실시간 회전 구현
시간의 흐름에 따라 카메라 위치를 원형 궤도로 이동시켜 회전 효과를 구현할 수 있다.

```cpp
const float radius = 10.0f;
float camX = sin(glfwGetTime()) * radius;
float camZ = cos(glfwGetTime()) * radius;

glm::mat4 view;
view = glm::lookAt(glm::vec3(camX, 0.0f, camZ), glm::vec3(0.0f, 0.0f, 0.0f), glm::vec3(0.0f, 1.0f, 0.0f));
```

### 키보드 입력 매핑
사용자의 키보드 입력에 반응하여 카메라 위치를 실시간으로 갱신한다.

```cpp
void processInput(GLFWwindow *window)
{
    const float cameraSpeed = 2.5f * deltaTime;
    if (glfwGetKey(window, GLFW_KEY_W) == GLFW_PRESS)
        cameraPos += cameraSpeed * cameraFront;
    if (glfwGetKey(window, GLFW_KEY_S) == GLFW_PRESS)
        cameraPos -= cameraSpeed * cameraFront;
    if (glfwGetKey(window, GLFW_KEY_A) == GLFW_PRESS)
        cameraPos -= glm::normalize(glm::cross(cameraFront, cameraUp)) * cameraSpeed;
    if (glfwGetKey(window, GLFW_KEY_D) == GLFW_PRESS)
        cameraPos += glm::normalize(glm::cross(cameraFront, cameraUp)) * cameraSpeed;
}
```

### 프레임 독립적 이동 (Delta Time)
하드웨어 성능에 관계없이 일정한 이동 속도를 유지하기 위해, 이전 프레임과의 시간 간격인 `deltaTime`을 활용하여 속도 계수를 조정한다.

```cpp
float currentFrame = glfwGetTime();
deltaTime = currentFrame - lastFrame;
lastFrame = currentFrame;
```

## 4. 마우스 제어 및 오일러 각도

### 오일러 각도 (Euler Angles)
3D 회전을 정의하기 위해 Pitch(X축 회전), Yaw(Y축 회전), Roll(Z축 회전) 값을 사용한다. 마우스 이동량에 따라 Pitch와 Yaw 값을 갱신하여 카메라의 시선을 제어한다.

### 시선 방향 벡터 산출
삼각함수를 활용하여 갱신된 오일러 각도로부터 최종 방향 벡터를 도출한다.

```cpp
direction.x = cos(glm::radians(yaw)) * cos(glm::radians(pitch));
direction.y = sin(glm::radians(pitch));
direction.z = sin(glm::radians(yaw)) * cos(glm::radians(pitch));
cameraFront = glm::normalize(direction);
```
